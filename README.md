Key Binds:

	S  	 	Decrease s value
	W 	 	Increase s value
	A 	 	Decrease b value
	D 	 	Increase b value
	Q 		Decrease r value
	E 		Increase r value
	ARROW KEYS  	Axis movement/Change view angle
	SPACEBAR 	Reset view angle and lorenz values back to default
	ESC 		Exit
